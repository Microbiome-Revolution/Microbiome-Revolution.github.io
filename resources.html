Project Resources

We are committed to open science. Below are links to GitHub repositories and tools that have been created or are used by the Microbiome Green Revolution project.

  • Sample Protocols — Sampling protocols for cataloguing and managing microbial samples.
  • Data Standards — Data collection and storage standards for researchers contributing new data to the project.
  • ARUA Workshop 2025 — A five-day workshop to unite ARUA scientists for training and collaboration in microbiome research, developed by MGR team members.
  • SimpleMetaPipeline — A metagenomic analysis pipeline, useful for analysing microbial communities collected by the MGR project.
  • The Multilingual Quantitative Biologist — Useful resources for all to learn about data science and ecoinformatics.

If you use these resources, please cite the corresponding publications or get in touch with us for collaboration opportunities.
